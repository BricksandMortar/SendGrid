Setup

Rock

Ideal Postcodes

To use this plugin you will need a Ideal Postcodes account. You can setup without payment with just an email address and password.

When you've created a new account you will be prompted to create a new API key, which will come with 15 free lookups to test the service. Before you are able to use the service you will need to authorise your account.
